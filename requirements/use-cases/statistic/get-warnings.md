# Summary

- **Title:** Category Statistics
- **Description:** System displays some informations about a category
- **Pre-condition:** Category must exists
- **Pos-condition:** Category statistics are shown

# Flow

### Normal Flow

1. User initiates the category statistics process.
2. The system displays informations, such as:
    - Number of entries
    - Number of done entries
    - Number of undone entries
    - Money total expected
    - Money total real
    - Money added
    - Money removed
    - Money saved
    - Money needed to be paid
