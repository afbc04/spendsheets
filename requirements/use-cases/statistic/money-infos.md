# Summary

- **Title:** Money Informations
- **Description:** System displays some informations about the money
- **Pre-condition:** -
- **Pos-condition:** Money informations are shown

# Flow

### Normal Flow

1. User initiates the money informations process.
2. The system displays informations, such as:
    - Initial money
    - Lost money
    - Money total expected
    - Money total real
    - Money added
    - Money removed
    - Total money on savings
    - Total money saved
    - Money needed to be paid
    - Money expected to be received
