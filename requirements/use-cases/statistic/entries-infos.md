# Summary

- **Title:** Entries Informations
- **Description:** System displays some informations about the entries
- **Pre-condition:** -
- **Pos-condition:** Entries informations are shown

# Flow

### Normal Flow

1. User initiates the entries informations process.
2. The system displays informations, such as:
    - Number of entries
    - Number of deleted entries _(Including percentage)_
    - Number of cancelled entries _(Including percentage)_
    - Number of ignored entries _(Including percentage)_
    - Number of draft entries _(Including percentage)_
    - Number of done entries _(Including percentage)_
    - Number of stalled entries _(Including percentage)_
    - Number of scheduled entries _(Including percentage)_
    - Number of accomplished entries _(Including percentage)_
    - Number of overdue entries _(Including percentage)_
    - Number of active entries _(Including percentage)_
    - Number of transation entries
    - Number of loan entries
    - Number of due payment entries
    - Number of saving entries
