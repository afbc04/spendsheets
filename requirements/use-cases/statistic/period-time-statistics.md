# Summary

- **Title:** Period of Time Statistics
- **Description:** System provides information about a certain period of time
- **Pre-condition:** Dates are valid
- **Pos-condition:** Statistics are shown

# Flow

### Normal Flow

1. User initiates the date statistics process.
2. The system provides informations about what happened in the period of time, such as:
    - Number of entries
    - Number of transation entries _(Including percentage)_
    - Number of loan entries _(Including percentage)_
    - Number of due payment entries _(Including percentage)_
    - Number of saving entries _(Including percentage)_
    - Real money added
    - Real money removed
3. The system also provides top N categories in this period of time, indicating:
    - ID of category
    - Name of category
    - Number of entries
    - Real money added
    - Real money removed
    - Percentage related to others categories
