# Summary

- **Title:** General Informations
- **Description:** System displays some general informations about the system
- **Pre-condition:** -
- **Pos-condition:** General informations are shown

# Flow

### Normal Flow

1. User initiates the general informations process.
2. The system displays informations, such as:
    - Number of entries
    - Number of transation entries _(Including percentage)_
    - Number of loan entries _(Including percentage)_
    - Number of due payment entries _(Including percentage)_
    - Number of saving entries _(Including percentage)_
    - Number of active monthly services
    - Number of inactive monthly services
    - Number of categories
    - Number of tags
    - Money total expected
    - Money total real
