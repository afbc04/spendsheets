# Summary

- **Title:** List Loan Entries
- **Description:** The system gives a list of the loans undone registered
- **Pre-condition:** -
- **Pos-condition:** Loan Entries undone are listed

# Flow

### Normal Flow

1. User indicates he wants to list categories
2. System list categories, providing informations such as:
    - ID
    - ID of Category
    - Date of last change
    - Status
    - Money need to be paid or to receive
