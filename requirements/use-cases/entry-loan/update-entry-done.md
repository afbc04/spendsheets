# Summary

- **Title:** Update Loan Entry as DONE
- **Description:** Administrators can edit existing loan entries' status
- **Pre-condition:** User must be Administrator & entry must exists
- **Pos-condition:** Entry's status is updated

# Flow

### Normal Flow

1. User initiates the update loan entry to DONE process.
2. User selects the entry
3. System updates entry's information, such as:
    - Update last change date
    - Use current date as finish date
    - Update status as DONE
