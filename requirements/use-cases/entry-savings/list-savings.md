# Summary

- **Title:** List Saving Entries
- **Description:** The system gives a list of the savings registered
- **Pre-condition:** -
- **Pos-condition:** Saving Entries are listed

# Flow

### Normal Flow

1. User indicates he wants to list categories
2. System list categories, providing informations such as:
    - ID
    - ID of Category
    - Date of last change
    - Status
    - Due Date
    - Money need to be paid
