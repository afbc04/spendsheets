# Summary

- **Title:** Edit Tag
- **Description:** Administrators can edit existing tags
- **Pre-condition:** User must be Administrator & tag must exists
- **Pos-condition:** Tag's information is updated

# Flow

### Normal Flow

1. User initiates the edit tag process.
2. User selects the tag
3. User provides the new information, such as:
    - Name of tag
    - Description of tag
4. System validates the information provided
5. System updates the tag

### Exception (4) [Information is invalid]

5. System informs to user that information provided is invalid
