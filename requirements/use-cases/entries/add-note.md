# Summary

- **Title:** Add Note to Entry
- **Description:** Administrators can add note to entry
- **Pre-condition:** User must be Administrator & entry must exists
- **Pos-condition:** Entry has a new note

# Flow

### Normal Flow

1. User initiates the add note to entry process.
2. User selects an entry
3. User provides the new note
4. System saves the note to entry, saving the note date as well
