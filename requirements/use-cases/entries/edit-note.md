# Summary

- **Title:** Edit Note of Entry
- **Description:** Administrators can edit notes of enties
- **Pre-condition:** User must be Administrator & entry must exists
- **Pos-condition:** Entry has a note changed

# Flow

### Normal Flow

1. User initiates the edit note to entry process.
2. User selects an entry
3. User selects the note
4. User provides the new information of note, such as:
    - Note
5. System saves the note of entry, updating the last change date of entry
